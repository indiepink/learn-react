1. 컴포넌트
 - UI를 구성하는 재사용 가능한 코드 조각

 - 파일명.jsx
 => rafce (react snippet 설치로 자동 완성 가능)


2. JSX의 규칙
 - class => className
 - <></>
 - HTML코드에서 JavaScript를 사용할 땐 중괄호
  => <div> {a} </div> , <input value = {b}/>
- 컴포넌트명은 PascalCase, 내부코드는 camelCase


3. State -내부
 - 리액트에 존재하는 특별한 변수
 - 두 개의 변수로 이루어져있음 (첫 번째: 데이터, 두 번째: 함수)

4. props -외부
 - 자식이 부모한테 데이터를 넘겨주는 것(데이터인 값을 넘겨 줌)

① Logical operators(단축 평가 논리 계산법)
 - and(&&), or(||), not(boolean으로 변경하려면 앞에 !!을 붙임)

② Object Destructuring(구조분해할당): 배열, 객체에서 각각의 값을 손쉽게 변수에 할당 하는 것

③ Spread syntax - 배열이나 객체를 개별 요소로 분해하거나 결합할 때 사용(복제 기능 O)


5. Router
 - yarn add react-router-dom@6 (cmd에서 설치)
 - package.json에서 "react-router-dom": "6" 확인
